# tstamp-utils

A simple Python utility to generate and print formatted timestamps.

## Installation

You can install this package directly from GitHub (once uploaded):

```bash
pip install git+[https://github.com/Dhaval-Sathawara/tstamp-utils.git](https://github.com/Dhaval-Sathawara/tstamp-utils.git)

## Usage

import tstamp

# Call the function
tstamp.print_timestamp()
